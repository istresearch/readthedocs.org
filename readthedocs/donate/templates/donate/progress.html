Welcome to the IST Research Read the Docs!

This place is a home for project documentation created by developers for developers. The documentation here should give any developer the ability to successfully learn and understand any project being worked on within the company. The goal of putting this together is to facilitate a centralized place for all developer documentation that everyone could benefit from seeing or having access to.

The technical documentation outlined within can contain API documentation, architecture overviews, UML diagrams, testing procedures, version history, or anything you think is relevant for both new and old developers to learn about and contribute to a specific project. Some of the benefits of putting your documentation here is that it integrates nicely with Github version control and repositories, can generate PDF's of your documentation, and can automatically build your documentation when you push your git commit.

For more information on how to get started, please see the General Overview Project.
